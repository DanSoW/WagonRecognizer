Название приложения: Информационная система для распознования номеров железнодорожных полувагонов

Перечень сторонних библиотек, используемых для работы приложения:
Tess4j, OpenCV, Gson, Apache, JavaFX

Описание приложения: Информационная система состоит из трёх модулей: "Центральный сервер",
"Пользователь" и "Камера". К серверной части принадлежит модуль "Центральный сервер", а 
к клиентской части модули "Пользователь" и "Камера". В модуле "Пользователь" реализуется
механизм взаимодействия с базой данных, а в логике модуля "Камера" механизм распознования номера
полувагона и его добавление в базу данных. Модуль "Центральный сервер" использует базу данных
MySQL для хранения данных, а также представляет интерфейс по протоколу HTTP, реализованный
с помощью POST/GET запроса.
